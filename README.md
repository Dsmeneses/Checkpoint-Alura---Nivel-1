Carreira Análise de Dados: Checkpoint Nível 1

Aqui contém todo o conteúdo desenvolvido no Checkpoint de carreira da dados da Alura.

Ferramentas Utilizadas: 
- Visual Studio (Para a visualização dos arquivos '.csv')
- Google Colab (Criação do Notebook) juntamente com Python voltado para análise de dados (Pandas, Matplotlib)
- Power BI (Criação das Telas)
- MySql (Criação de consultas e tabelas)

Projeto:

A ZOOP Megastore está buscando otimizar suas operações de e-commerce através de uma análise detalhada de dados. 
O objetivo é utilizar ferramentas de análise de dados para extrair insights valiosos que possam melhorar a gestão de estoque,otimizar campanhas de marketing e aumentar a eficiência.

A equipe que você integra será responsável por realizar análises exploratórias, criar visualizações e desenvolver relatórios que ajudem na tomada de decisões estratégicas.

O primeiro trimestre do ano acabou de fechar e a diretoria está ansiosa por insights para planejar o próximo semestre. 
Seu desafio é conduzir uma análise completa dos dados de vendas e clientes para responder a uma pergunta central: "Onde devemos focar nossos esforços de marketing e gestão de estoque para maximizar o lucro e a satisfação do cliente?"

Este projeto testará sua capacidade de transformar dados brutos em inteligência de negócio, utilizando as principais ferramentas do dia a dia de um analista.

O projeto foi desenvolvido em 4º etapas, sendo elas em ordem:
1ª etapa - Exploração e limpeza dos dados com Pandas --> Feito com Google Colab e Python
2ª etapa - Estruturando os dados com MySQL --> Criação e estruturação das tabelas, juntamente com consultas.
3ª etapa - Análise geográfica e de clientes com Power BI --> Mapa dos estados com o faturamento, Faturamento dos clientes regulares e Assinantes, e Categorias de produtos que mais venderam.
4ª etapa - dashboard estratégico e recomendações finais --> Telas comparativas, recomendações de estratégia.

Dificuldades Encontradas:

- Tive dificuldades em fazer a importação dos dados dos arquivos '.csv' para o MySQL, principalmente na parte de acentuação, tentei arrumar de diversas formas, porém sem sucesso. Fiz a inserção deles manualmente, como consta no arquivo na aba 'MySQL'.

- Outra dificuldade foi em relação aos mapas dentro do Power BI, os Estados presentes estão em siglas, e o de estado de Santa Catarina (SC), estava referenciando o estado de South Carolina(Carolina do Sul) nos EUA. Tentei arrumar de todo modo, porém, mesmo seguindo várias dicas, não foi possível.
